# 幸福人 Classic V1.2

更新重點：
- 回字型棋盤
- 經過起點／發薪日自動領薪
- 外圈擲 2 顆骰子，職業道路擲 1 顆骰子
- 八大職業道路雛形
- 玩家人生口袋：現金、機會卡、經驗卡
- 機會卡可儲存、使用、出售
- 經驗卡可取代擲骰
- 只能操作目前回合玩家的口袋
- 達成秘密幸福目標立即結束遊戲

安裝：
npm install
npm run dev


## V1.2 更新重點
- 防止手機／觸控板下滑造成遊戲重置：加入 localStorage 自動存檔與重開確認。
- 最近紀錄改為右側欄置頂，方便即時查看。
- 支援背景音樂：請將 `bgm.wav` 放在 `public/` 資料夾，遊戲內按「播放音樂」。
- 棋子移動改成一步一步前進，避免擲骰後瞬間跳太遠找不到角色。
