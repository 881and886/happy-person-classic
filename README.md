# 幸福人 Classic 2.0 Alpha

## 本版重點
- 外圈人生主道路 + 八大職業內圈
- 外圈雙骰、內圈單骰
- 每條職業內圈至少 12 格
- 玩家人數 1～6 人
- 動物棋子選擇
- 財富／快樂／名譽目標設定與隱藏判定
- 玩家人生口袋：現金、機會卡、經驗卡
- 機會卡可保留、使用、出售
- 經驗卡可替代擲骰
- Recent Log 置頂
- BGM 開關：請把 bgm.wav 放到 public/bgm.wav
- 棋子逐格移動動畫
- 經過發薪日自動發薪
- 達成預定幸福目標即結束

## 安裝
npm install
npm run dev

## 更新方式
若你已有 Vite React 專案，可直接覆蓋：
- package.json
- index.html
- src/main.jsx
- src/style.css
- public/README-bgm.txt

若已有自己的 bgm.wav，請放在：
public/bgm.wav
