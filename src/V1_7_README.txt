幸福人小遊戲 V1.7 更新內容

1. 新增八大職業內圈地圖：學院、企業、農墾、政治、航海、電影明星、開礦、月球探險。
2. 新增入口／出口機制：走到外圈入口時，可選擇進入職業道路或留在外圈。
3. 新增內圈單骰：外圈擲兩顆骰子，顯示 2–12；內圈擲一顆骰子，顯示 1–6。
4. 新增專屬事件池：每條職業路線有自己的事件內容與金錢變化。
5. 修正 UI 流程：中央大字顯示骰子點數，下方顯示走幾步、抵達哪裡、發生什麼事。
6. 修正使用退休權去度假：按鈕會使玩家下次輪到時休息一回合。
7. 修正宣告破產：按鈕會讓玩家資產歸零並退出本局。

使用方式：
將 App_V1_7.jsx 改名為 App.jsx，覆蓋你專案 src/App.jsx 後重新部署即可。
